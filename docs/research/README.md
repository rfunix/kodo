# Kōdo Language Research

Relatórios de pesquisa de mercado e tendências de linguagens de programação,
gerados automaticamente pelo Kōdo Architect agent.

## Formato

Cada relatório segue o padrão: `YYYY-MM-DD-topic.md`

## Tópicos Monitorados

- Linguagens compiladas: Rust, Zig, Carbon, Mojo, Vale, Gleam, Roc, Austral
- Verificação formal: Dafny, F*, Lean, Idris
- AI-assisted programming e linguagens para agentes
- Técnicas de compilação: MLIR, Cranelift, LLVM
- Type systems e teoria de PLs
