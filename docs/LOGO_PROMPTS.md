# Prompt Midjourney — Logo Kōdo

## Contexto

A linguagem Kōdo (コード) precisa de uma identidade visual (logo) para o README e branding geral. O logo deve capturar a essência da linguagem: **"O Caminho do Código"** — uma linguagem compilada projetada para agentes de IA, com foco em transparência, correção por construção, contratos formais e auditabilidade total.

## Conceitos visuais centrais

- **道 (dō) — O Caminho**: disciplina, maestria, caminho claro e sem ambiguidade
- **Transparência/Cristal**: tudo visível, sem caixas pretas
- **Correntes/Encadeamento**: certificados encadeados, trust chains
- **Geometria precisa**: zero ambiguidade, LL(1), cada construção tem exatamente uma interpretação
- **Ponte entre máquina e humano**: projetado para agentes IA, auditável por humanos

## Dicas de uso

- Gere várias variações de cada prompt e escolha a melhor
- Adicione `--seed [número]` para reproduzir resultados que gostar
- Use `--stylize 50` para resultados mais literais ou `--stylize 750` para mais artísticos
- Após escolher o logo, use `/describe` no Midjourney para refinar o prompt
- Para versão com texto "Kōdo", gere separadamente e combine em editor
